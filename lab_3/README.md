# Лабораторная работа 3 — Rust: распознавание фигуры

Нейросеть распознаёт **квадрат, прямоугольник, круг, треугольник** на сетке 5×5 из **0** и **1**.

## Запуск

```sh
cd lab_3
cargo run
```

## Выход

На первой строке ввода введите `q`.

## Классы фигур

| Класс | Пример (5 строк по 5 цифр) |
|-------|----------------------------|
| `square` | `00000` `01110` `01110` `01110` `00000` |
| `rectangle` | `00000` `11111` `11111` `00000` `00000` |
| `circle` | `01110` `10001` `10001` `10001` `01110` |
| `triangle` | `00100` `01110` `11111` `00000` `00000` |

## Как вводить

5 строк по 5 символов `0` или `1`, без пробелов.

## Пример ответа

```
Result: circle
Scores: square=0.012, rectangle=0.008, circle=0.971, triangle=0.009
```
