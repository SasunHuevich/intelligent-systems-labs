# Лабораторные работы по предмету "Интеллектуальные системы" (Магистратура 1 курс)

Реализованы четыре работы по темам: логическое программирование (Prolog), нейронные сети (Rust) и экспертные системы (CLIPS).

---

## Лабораторная работа 1 – Prolog: факты и правила

**Цель:** научиться описывать базу знаний (факты и правила) и выполнять запросы.

**Описание:**  
- **Вариант 1 (lab_1.1.pl)** – животные, их цвета и владельцы. Определены правила для собаки, домашнего животного и отношения владения.  
- **Вариант 5 (lab_1.5.pl)** – родственное дерево; заданы отношения родитель, ребёнок, предок, мать.

--- 

## Лабораторная работа 2 – Prolog: списки

**Цель:** освоить рекурсивную обработку списков.

**Описание:**
- **Вариант 4 (lab_2.4.pl)** – пословный перевод с английского на французский.
- **Вариант 8 (lab_2.8.pl**) – поиск последнего элемента списка

---

## Лабораторная работа 3 – Rust: распознавание фигур

**Цель:** реализовать простую нейросеть (скрытый слой, сигмоида, backpropagation) для классификации геометрических фигур на сетке 5×5.

**Описание:**
Сеть распознаёт четыре класса: square, rectangle, circle, triangle. Ввод – 5 строк по 5 символов (0 или 1). Выход – предсказанный класс и уверенность (scores).

---

## Лабораторная работа 4 – CLIPS: экспертная система

**Цель:** разработать продукционную экспертную систему для диагностики неисправностей ПК.

**Описание:**
Система последовательно задаёт вопросы о симптомах (включение, индикаторы, кулер, загрузка, периферия, диск, сеть, монитор, звук, клавиатура, мышь, БП, перегрев, производительность) и на основе ответов выдаёт рекомендации по устранению неисправности. База знаний реализована с помощью правил CLIPS.
