# Лабораторная работа — Prolog: списки

Рекурсивная обработка списков. Два отношения из индивидуального задания.

## Запуск

```sh
swipl lab_2.4.pl
swipl lab_2.8.pl
```

## Выход из консоли swipl

```sh
halt.
```

## Вариант 4

`lab_2.4.pl`

Пословный перевод предложения (список слов) с английского на французский.

### Запросы и ответы

| Запрос | Результат |
|--------|-----------|
| `translate([hello, world]).` | **[bonjour, monde].** **true.** |
| `translate([i, love, you]).` | **[je, aime, tu].** **true.** |
| `translate([dog, cat]).` | **[chien, chat].** **true.** |
| `translate([]).` | **[].** **true.** |

### Интерпретации `translate/1`

**Декларативная:** «Предложение Words переводится по словарю и выводится как список французских слов».

**Процедурная:** `translate_list/2` собирает перевод, `write/1` печатает результат.

## Вариант 8

`lab_2.8.pl`

Последний элемент списка.

### Запросы и ответы

| Запрос | Результат |
|--------|-----------|
| `last(X, [a, b, c, d]).` | **X = d.** |
| `last(X, [hello]).` | **X = hello.** |
| `last(X, []).` | **false.** |

### Интерпретации `last/2`

**Декларативная:** «X — последний элемент списка List».

**Процедурная:** если список `[H|T]` и `T` не пустой — ищем последний в `T`; если список `[X]` — ответ `X`.
